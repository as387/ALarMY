
1. Краткое описание бота:

Этот бот предназначен для создания напоминаний о событиях, встречах, задачах и других важных делах. Он будет напоминать вам о предстоящих событиях и давать возможность подтверждать их. Вы можете настроить время, через которое будет запрашиваться подтверждение, и другие параметры.

2. Все команды:

/help - Получение инструкции в PDF формате.
/set_confirmation_interval - Изменение времени для подтверждения напоминаний.
/add_reminder - Добавление нового одноразового напоминания.
/set_repeating_reminder - Добавление повторяющегося напоминания с выбором периода повторений.
/manage_reminder - Управление напоминаниями (включение/выключение подтверждения).
/list_reminders - Показать все напоминания.
/delete_reminder - Удаление напоминания.

3. Все сюжеты добавления напоминаний:

3.1. Добавить одноразовое напоминание:
- Введите команду `/add_reminder`.
- Бот запросит время напоминания (например, 07:30).
- Затем бот запросит описание события (например, Завтрак).
- Напоминание будет добавлено и сохранено.

Пример:
```
/add_reminder
Время: 07:30
Событие: Завтрак
```

3.2. Добавить повторяющееся напоминание:
- Введите команду `/set_repeating_reminder`.
- Бот запросит время напоминания (например, 18:00).
- Затем бот запросит описание события (например, Прогулка).
- После этого бот попросит выбрать период повторений (ежедневно, еженедельно и т.д.).
- Напоминание будет добавлено с выбранным интервалом повторения.

Пример:
```
/set_repeating_reminder
Время: 18:00
Событие: Прогулка
Период повторений: ежедневно
```

3.3. Удаление напоминаний:
- Введите команду `/delete_reminder`.
- Бот покажет список всех ваших напоминаний с номерами.
- Выберите номер напоминания, которое нужно удалить.

Пример:
```
/delete_reminder
Удалить напоминание номер: 2
```

3.4. Установка подтверждения:
- Введите команду `/manage_reminder`.
- Бот покажет список ваших напоминаний.
- Укажите номера напоминаний, для которых нужно включить или выключить подтверждение.

Пример:
```
/manage_reminder
Включить подтверждение для напоминания номер: 1
```

4. Баги и что с ними делать:

4.1. Сломался сюжет:
- Если бот не правильно обрабатывает команду или не выполняет нужные действия, попробуйте перезапустить бота. Если проблема сохраняется, сообщите администратору.

4.2. Зависла кнопка:
- Если кнопка не реагирует на нажатие, попробуйте отправить команду снова или перезапустите бота. Это может быть связано с техническими проблемами или сбоем на сервере.

4.3. Бот не отвечает на команды:
- Если бот не отвечает на команды, убедитесь, что ваше интернет-соединение стабильно. Попробуйте отправить команду снова. Если это не помогает, перезапустите бота или сообщите об ошибке разработчику.

4.4. Проблемы с напоминаниями:
- Если вы не видите или не можете изменить напоминания, убедитесь, что вы правильно вводите команды и данные. Если ошибка продолжает возникать, сообщите об этом через команду `/help` для получения дополнительных инструкций.
